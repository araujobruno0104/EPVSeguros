# Site EPV Corretora de Seguros

Projeto de landing page premium para a EPV Corretora de Seguros LTDA.

## Arquivos

- `index.html` - estrutura da página
- `styles.css` - estilo visual
- `script.js` - menu mobile e envio de cotação para WhatsApp

## Personalização rápida

1. Coloque a logo da EPV no caminho `assets/logo-epv.png`.
2. Troque o número de WhatsApp:
   - em `index.html` (link do botão flutuante)
   - em `script.js` (variável `whatsappNumber`)
3. Atualize telefone, e-mail e demais informações em `index.html`.

## Como executar

Basta abrir o arquivo `index.html` no navegador.
